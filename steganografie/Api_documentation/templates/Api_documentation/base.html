About Steganography

Steganography is the practice of concealing information within another message or physical object to avoid detection. Steganography can be used to hide virtually any type of digital content, including text, image, video, or audio content. That hidden data is then extracted at its destination.

About this project

This is my final project for the end of the IT School course. This project consists of encrypting a message in a photo and being able to decrypt that message.

The rules are very simple:

  1. We click on Hide in the nav bar, and in Hide we have to upload a photo with .jpg extension.
  2. We have to add a message we want, and click on the hide button.
  3. After clicking on the submit button we will be informed that the photo has been successfully encrypted and we will be redirected to the main site.
  4. With the encrypted photo in the database, we can click on Decrypt in the navbar and upload the photo from the hidden_images folder. After uploading the photo by clicking on the Decrypt button we will get the secret message.
